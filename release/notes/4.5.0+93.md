# Icarus 4.5.0 Patch Notes

Released July 18, 2026.

## Highlights

- View cones now stop at walls and account for different map elevations, making vision previews more accurate.
- Agents now navigate around map geometry during page transitions for more natural movement between setups.
- Library sorting and drawing tool preferences are now remembered between sessions.
- Resizing abilities and agent markers now keeps them centered on their intended map position.

## Additional changes

- Added a thinner drawing stroke option.
- Agents placed inside lineup groups now appear in strategy and folder previews.
- Truncated strategy and folder names reveal their full names on hover.
- Refined the spacing and alignment of map selectors, strategy cards, delete controls, and the quick strategy switcher.
- Fixed native text-editing shortcuts such as paste and undo being intercepted by application shortcuts.
- Fixed preference changes occasionally overwriting one another when several settings were changed quickly.
- Fixed live marker-size changes sometimes not marking the strategy as unsaved.
- Fixed the quick strategy switcher's positioning and tooltip behavior.
- Fixed empty tooltip bubbles on custom-shape buttons.
- Fixed full-name tooltips appearing when text was not truncated.
- Improved page-transition performance and state refresh after map geometry loads.
- Improved view-cone clipping around compound walls, separate floor contours, coordinate transformations, and multi-level areas.
- Fixed desktop update staging so stale files from earlier builds are removed before packaging.

Internal build, test, workflow, and development-tool changes are intentionally omitted.
